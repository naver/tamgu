## First Program in Tamgu

To provide a clearer understanding, here is a small program that displays the content of a string.

### Declaration
```tamgu
string s;
int i;
```

### Instantiation
```tamgu
s = "abcd";
i = 100;
```

### Display
```tamgu
print("S=", s, " I=", i, "\\n");
```

### Execution
```
S=abcd  I=100
```
