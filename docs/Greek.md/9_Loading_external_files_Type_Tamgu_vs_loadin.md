## Φόρτωση Εξωτερικών Αρχείων: Τύπος Tamgu εναντίον loadin

Ο τύπος `tamgu` χρησιμοποιείται για τη δυναμική φόρτωση ενός συγκεκριμένου προγράμματος Tamgu. Από την άλλη, το `loadin` χρησιμοποιείται για τη φόρτωση του περιεχομένου ενός αρχείου στο τρέχον πρόγραμμα.

### Μέθοδοι

1. `tamgu var(string Tamgu pathname)`: Δημιουργεί και φορτώνει ένα πρόγραμμα Tamgu.

### Εκτέλεση Εξωτερικών Συναρτήσεων

Οι συναρτήσεις που είναι διαθέσιμες στο αρχείο Tamgu μπορούν να κληθούν μέσω μιας μεταβλητής `tamgu`.

#### Παράδειγμα

```cpp
// Στο πρόγραμμά μας test.tmg, υλοποιούμε τη συνάρτηση: Read
// Στο αρχείο test.Tamgu
function Read(string s) {
    return (s + "_toto");
}
//-------------------------------------------
// Στο αρχείο call.Tamgu
// Στο πρόγραμμα κλήσης, φορτώνουμε πρώτα το test.Tamgu, και στη συνέχεια εκτελούμε τη συνάρτηση Read
tamgu kf('c:\test.tmg'); // Φορτώνουμε ένα πρόγραμμα που υλοποιεί τη συνάρτηση Read
string s = kf.Read("xxx"); // Μπορούμε να εκτελέσουμε τη συνάρτηση Read στο τοπικό μας πρόγραμμα.
```

### Ιδιωτικές Συναρτήσεις

Εάν δεν θέλετε εξωτερικά προγράμματα να έχουν πρόσβαση σε συγκεκριμένες συναρτήσεις, μπορείτε να τις προστατεύσετε δηλώνοντάς τις ως ιδιωτικές.

#### Παράδειγμα

```cpp
// Υλοποιούμε μια συνάρτηση, η οποία δεν μπορεί να κληθεί από εξωτερικά
private function CannotBeCalled(string s) { ... }
```

### `loadin(string pathname)`

Ο Tamgu παρέχει επίσης έναν άλλο τρόπο φόρτωσης αρχείων σε ένα άλλο πρόγραμμα. Το `loadin(pathname)` φορτώνει το περιεχόμενο ενός αρχείου στο τρέχον πρόγραμμα. Με άλλα λόγια, το `loadin` λειτουργεί ακριβώς όπως αν το τρέχον αρχείο περιείχε τον κώδικα αυτού που φορτώνεται με το `loadin`.

Ο κώδικας φορτώνεται πραγματικά στην ίδια θέση όπου γράφεται η κλήση του `loadin`.

#### Παράδειγμα

```cpp
// Στο πρόγραμμά μας test.tmg, υλοποιούμε τη συνάρτηση: Read
// Στο αρχείο test.Tamgu
function Read(string s) {
    return (s + "_toto");
}
//-------------------------------------------
// Στο αρχείο call.Tamgu
// Στο πρόγραμμα κλήσης, φορτώνουμε πρώτα το test.Tamgu, και στη συνέχεια εκτελούμε τη συνάρτηση Read
loadin('c:\test.tmg'); // Φορτώνουμε τον κώδικα που υλοποιεί τη συνάρτηση Read
string s = Read("xxx"); // Μπορούμε να εκτελέσουμε τη συνάρτηση Read απευθείας στο τοπικό μας πρόγραμμα.
```
