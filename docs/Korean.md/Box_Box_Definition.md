## 상자 (상자 정의)

이 유형은 주 창에 텍스트와 함께 상자를 그리는 데 사용됩니다. 다음과 같은 메서드를 노출합니다:

### 메서드

1. `create(int x, int y, int w, int h, string label)`: 레이블이 있는 상자를 생성합니다.
2. `type(int boxtype)`: 상자 유형을 수정합니다 (아래 상자 유형 목록 참조).
