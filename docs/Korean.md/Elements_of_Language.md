## 언어 요소

Tamgu 프로그램은 변수 선언, 함수 선언 및 프레임(또는 클래스) 선언으로 구성됩니다. 변수는 어디에서든 선언할 수 있으며, 함수도 루프를 제외한 모든 곳에서 선언할 수 있습니다.

### 주석

한 줄 주석은 `//`로 시작합니다.

```javascript
// 이것은 주석입니다.
```

여러 줄 주석은 `/@`와 `@/` 사이에 작성됩니다.

```javascript
/@ 
이것은 여러 줄 주석입니다.
@/
```

### 함수

함수는 `function` 키워드로 선언되며, 이름과 매개변수가 뒤에 옵니다.

### 프레임

프레임은 `frame` 키워드로 선언되며, 이름이 뒤에 옵니다. 하위 프레임은 그냥 다른 프레임 내에서 프레임으로 선언됩니다.

### 함수 및 프레임 사전 선언

Tamgu에서는 함수와 프레임의 사전 선언이 필요하지 않습니다. 인터프리터는 먼저 코드를 반복하여 모든 함수와 프레임을 감지하고 미리 선언합니다.

따라서 다음 코드는 유효합니다:

```javascript
// call1에서 call2를 호출합니다.
function call1(int x, int y) { 
    call2(x, y);  
} 

// call1 이후에 call2가 선언됩니다.  
function call2(int x, int y) { 
    println(x, y);  
}
```
