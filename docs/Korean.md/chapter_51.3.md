# 비트맵

이 유형은 창이나 버튼에서 표시할 수 있는 비트맵 이미지를 정의하는 데 사용됩니다. 다음과 같은 하나의 특정한 메소드를 노출합니다:

## 메소드
1. `load(ivector bm, int w, int h)`: ivector에서 비트맵 이미지를 w와 h의 크기로 로드합니다.
